# Mini Social Media Platform

## Overview

Mini Social Media Platform is a web application that allows users to register, create posts, like posts, comment on posts, and follow other users.

## Features

* User Registration
* User Profiles
* Create Posts
* View Posts
* Like Posts
* Add Comments
* Follow Users
* MySQL Database Integration

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MySQL

## Database Tables

* users
* posts
* comments
* likes
* followers

## Installation

1. repository link 

```bash
link:  https://github.com/Kalsoom-Gill/CodeAlpha_socialmediaplateform.git
```

2. Install dependencies

```bash
npm install
```

3. Import database.sql into phpMyAdmin

4. Start the server

```bash
node server.js
```

5. Open in browser

```text
http://localhost:3000
```
